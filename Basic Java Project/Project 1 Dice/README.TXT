PROJECT TITLE:      Simple Board Game
PURPOSE OF PROJECT: The aim of your program is to simulate a simple Board 
                    Game commonly played among children, where the players 
                    take turn to roll a dice, and move their game tokens 
                    on a physical board. The winner is the one who reaches 
                    a final position on the board first. 
VERSION or DATE:    1.0 (Mar. 28, 2017)
HOW TO START THIS PROJECT: Construct object of class Game, run the void
                           playGame() method.
AUTHORS:            Siyang Feng
USER INSTRUCTIONS:  Enter '1' to start a new game and set players. 
                    Enter '2' to roll dice and get new position. 
                    Be careful of penalty if your position is 11,22,33,44. 
                    The winner is the one who arrive 50 first. 
                    Enter '5' to exit game.\nLet's start!
